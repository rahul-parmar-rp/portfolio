# Draft Plan: Offline-First Tools and Datasets Directory

## Idea Summary

Build a curated directory of useful offline tools, extensions, software, datasets, and local-first project ideas.

## Problem

Offline-capable tools and datasets are scattered and hard to evaluate.

## Core MVP

- Categorized tool listings
- Capability tags and offline score
- Setup guides and starter templates
- Regularly updated recommendations

## Categories

- Productivity tools
- Offline databases
- Local intranet and mesh tools
- Developer tooling and scripts

## Milestones

1. Taxonomy and schema
2. Initial curated list
3. Search and filter UI
4. Community contribution workflow
