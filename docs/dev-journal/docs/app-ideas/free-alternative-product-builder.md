# Draft Plan: Free Alternative Product Builder

## Idea Summary

Identify pain points in existing products and build free alternatives with improved features users actually want.

## Problem

Users pay for tools that miss key quality-of-life features.

## Core MVP

- Problem discovery and feature gap mapping
- Fast prototype of better alternatives
- User feedback loop and voting
- Open-source first release model

## Strategy

- Focus on one painful feature gap at a time
- Launch lightweight but usable alternatives
- Keep transparent roadmap and public issues

## Milestones

1. Feature-gap database
2. First alternative MVP
3. User feedback dashboard
4. Open-source community growth
