# Draft Plan: Pi Browser Task Automation (Research)

## Idea Summary

Research a legal and terms-compliant automation assistant for repetitive in-app actions, focused on user productivity and reminders.

## Important Note

This concept should only proceed if platform terms explicitly allow automation. Do not bypass security, anti-bot controls, or user protections.

## Problem

Users repeat the same routine taps and check-ins daily.

## Core MVP

- Reminder-based workflow for daily actions
- Optional guided assistant (manual confirmation required)
- Local activity logs and completion streaks

## Safety and Policy

- Human-in-the-loop confirmations
- No credential capture
- No unauthorized automated clicking
- Strict compliance with platform terms

## Milestones

1. Terms and policy feasibility review
2. Reminder and checklist app
3. Optional accessibility-assisted flow (if compliant)
4. Audit and risk review
