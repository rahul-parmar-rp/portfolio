# Draft Plan: Corporate Holiday and Leave Optimizer

## Idea Summary

Build a leave planning tool that suggests maximum consecutive break days using public holidays and weekend bridges.

## Problem

Employees miss better leave planning opportunities due to fragmented calendars.

## Core MVP

- Country and region holiday sync
- Company holiday calendar input
- Leave bridge suggestions with scoring
- Team conflict view

## Value Add

- Best leave windows by month
- Policy-aware leave calculations
- Team coverage recommendations

## Milestones

1. Holiday data model
2. Optimization engine
3. Personal planner UI
4. Team and manager views
