# Draft Plan: Virtual Teen Patti Local Network Game

## Idea Summary

Build a virtual Teen Patti or Ace card game using Expo and React Native for hotspot and local Wi-Fi play.

## Problem

Players want quick in-person multiplayer card sessions without server setup.

## Core MVP

- Host and join game over local network
- Real-time card dealing and turn management
- Basic betting chips and round state
- Reconnect and session recovery

## Technical Notes

- Local lobby discovery
- WebSocket or local peer sync
- Rule validation on host device
- Anti-cheat and tamper checks

## Milestones

1. Local room engine
2. Core Teen Patti gameplay
3. UI polish and animations
4. Offline stats and replay logs
