# Draft Plan: Git and GitHub Automation Knowledge Base

## Idea Summary

Create a practical docs and script library for common Git and GitHub workflows.

## Problem

Teams repeat the same Git fixes and command sequences without standard references.

## Core MVP

- Curated shell scripts for common Git workflows
- Step-by-step runbooks with failure recovery notes
- Script safety checks and dry-run mode
- Searchable command catalog by use case

## Priority Topics

- Branch cleanup
- Safe rebasing
- PR prep and review checks
- Tagging and release scripts

## Milestones

1. Script catalog structure
2. Docs for top 20 workflows
3. Validation scripts and tests
4. Internal publishing site
