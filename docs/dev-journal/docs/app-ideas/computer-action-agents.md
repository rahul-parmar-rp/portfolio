# Draft Plan: Computer Action Agents

## Idea Summary

Build desktop agents that perform routine computer actions across browser, files, and productivity tools.

## Problem

Knowledge workers repeatedly perform multi-step tasks that could be agent-assisted.

## Core MVP

- Agent templates for common routines
- Natural language command to action plan
- Safe simulation mode before execution
- Logs, undo actions, and policy controls

## Use Cases

- Daily startup workflows
- Report packaging and sharing
- Browser data extraction
- Local file transformations

## Milestones

1. Agent runtime and planner
2. Tool connectors
3. Safety and sandboxing
4. Team deployment controls
