# Google Identity Services Demo

This page turns the original GIS micro-site into a live docs demo. The important part is not just the setup checklist; it is the client ID initialization flow, account state, and JWT inspection UI.

<GoogleIdentityServicesDemo />

## What This Version Demonstrates

- saving a client ID before initialization
- the difference between button render state and account state
- decoded JWT identity payloads
- why GIS is ideal for authentication-only flows

## Why It Differs From The OAuth Manager

This flow is about identity, not Gmail or Drive access. You get a signed ID token with user profile information, not a broad access token for Google APIs.
