# Google Multi-Account OAuth Manager

This was the most operational of the original Google micro-sites, so it needed a real interactive rebuild. The docs page now exposes the same shape: account onboarding, scope bundles, and service actions across Gmail, Drive, and Photos.

## Preserved Service Areas

- Gmail labels, drafts, and profile checks
- Drive storage quota lookup
- Google Photos albums and media previews
- full-library fetch caveats for personal testing setups

## Why The Photos Section Still Matters

The original demo carried a lot of value because it encoded the post-2025 Photos API reality. This interactive docs version keeps that operational model visible without depending on live network credentials.
