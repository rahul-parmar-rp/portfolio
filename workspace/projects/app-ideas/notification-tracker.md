# Draft Plan: Notification Tracker

## Idea Summary

Build an app that tracks, classifies, and summarizes mobile notifications for productivity and incident history.

## Problem

Important notifications get buried and users cannot easily audit what they missed.

## Core MVP

- Notification capture timeline
- App/source-based filtering
- Priority tags and custom labels
- Daily digest summary

## Advanced Features

- Rule-based auto categorization
- Spam or noise detection
- Searchable history with export
- Action reminders for missed alerts

## Milestones

1. Local notification logger
2. Filter and search interface
3. Summary and digest engine
4. Backup and export features
