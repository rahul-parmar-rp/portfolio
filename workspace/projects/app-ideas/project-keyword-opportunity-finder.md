# Draft Plan: Project Keyword Opportunity Finder

## Idea Summary

Build a tool that identifies high-opportunity project keywords and maps them to buildable product ideas.

## Problem

Good product opportunities are hidden in scattered trend and keyword data.

## Core MVP

- Keyword trend ingestion
- Difficulty and competition scoring
- Project idea suggestions by niche
- Weekly opportunity digest

## Outputs

- Build-now keyword lists
- Opportunity rationale cards
- Suggested MVP scope templates

## Milestones

1. Data source connectors
2. Scoring engine
3. Idea generation module
4. Ranking dashboard
