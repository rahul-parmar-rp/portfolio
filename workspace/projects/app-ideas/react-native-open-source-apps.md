# Draft Plan: React Native Open Source Apps Research

## Idea Summary

Create a curated research catalog of high-quality open-source React Native apps for learning and reuse.

## Problem

Developers waste time searching for reliable production-grade examples.

## Core MVP

- Curated list by app category
- Quality scoring rubric
- Architecture and stack notes
- Reusable module extraction suggestions

## Research Categories

- UI and design systems
- Offline-first apps
- Social and content apps
- Utility and productivity tools

## Milestones

1. Build scoring criteria
2. Collect and evaluate top repositories
3. Publish structured findings
4. Track updates and maintenance status
