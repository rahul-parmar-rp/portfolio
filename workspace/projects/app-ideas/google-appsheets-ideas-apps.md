# Draft Plan: Google AppSheet Ideas Apps

## Idea Summary

Build a set of lightweight business apps rapidly using Google AppSheet for operations, reporting, and approvals.

## Problem

Small teams need internal tools quickly but do not have full engineering bandwidth.

## App Concepts

- Inventory and stock tracker
- Field visit and service log app
- Expense approval workflow
- Daily sales and collection tracker
- Employee attendance and leave requests

## Core MVP

- Shared data model in Google Sheets
- AppSheet UI for create, update, and approvals
- Role-based views for manager and staff
- Notifications on status changes

## Milestones

1. Prioritize top 3 AppSheet use cases
2. Define common schema and naming conventions
3. Build pilot app templates
4. Add automation and usage analytics
