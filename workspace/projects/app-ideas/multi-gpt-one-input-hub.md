# Draft Plan: Multi-GPT One Input Hub

## Idea Summary

Build a single interface that sends one prompt to multiple AI assistants and compares results in one screen.

## Problem

Users switch across multiple AI tools repeatedly to evaluate output quality.

## Core MVP

- Single prompt input
- Multi-provider parallel query dispatch
- Unified response panel and comparison view
- Save and share conversation snapshots

## Providers

- ChatGPT
- Claude
- Gemini
- Additional pluggable providers

## Core Controls

- Token and cost tracking
- Rate limit handling and retries
- Privacy-safe request routing
- Response ranking filters

## Milestones

1. Provider connector layer
2. Unified UI and comparison tools
3. Prompt history and exports
4. Team collaboration mode
