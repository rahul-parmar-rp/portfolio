# Draft Plan: React Questions Problem Poster Workflow for Twitter Engagement

## Idea Summary

Build a workflow that generates React coding questions and posts them to social channels to drive engagement.

## Problem

Consistent educational posting is hard to maintain without a repeatable creation and publishing process.

## Core MVP

- Question bank and difficulty levels
- Auto-generate daily React problem posts
- Publish queue for Twitter/X
- Engagement tracking and topic iteration

## Workflow Stages

1. Topic selection
2. Question generation and review
3. Scheduled posting
4. Reply and follow-up prompt generation

## Milestones

1. Content template engine
2. Scheduler integration
3. Analytics and scoring dashboard
4. Multi-platform expansion
