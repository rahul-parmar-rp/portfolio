# ChatGPT Login Popup Blocker

## Description

This Chrome extension blocks the login popup for users who are not signed in, allowing for seamless access to the application without interruptions.

## Plan

- Research existing popup blockers in Chrome.
- Develop the extension to intercept and block specific popup requests.
- Implement a user-friendly options page to enable or disable the blocker.
- Test the extension extensively for edge cases.
- Publish the extension to the Chrome Web Store.
