# Draft Plan: OTP Cleaner Using Offline AI

## Idea Summary

Build an offline-first OTP and SMS cleaner app that classifies messages locally and suggests cleanup actions.

## Problem

OTP and transactional SMS clutter inboxes, making important communication hard to find.

## Core MVP

- Local SMS ingestion and parsing
- OTP detection and extraction
- Auto-tagging by category
- Safe cleanup workflow with confirmations

## Offline AI Scope

- On-device classification model
- No cloud upload of personal SMS content
- Lightweight model optimized for mobile

## Milestones

1. SMS parser and rules engine
2. Offline classification model integration
3. Batch actions and safeguards
4. Explainability and confidence indicators
