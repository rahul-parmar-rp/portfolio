# Draft Plan: Permanent Roommates

## Idea Summary

Create a secure roommate matching app with verified identity checks and same-gender roommate matching (male-to-male, female-to-female).

## Problem

Finding safe and trustworthy roommates is difficult, especially in cities with frequent migration.

## Target Users

- Students
- Working professionals
- Property owners and brokers

## Core MVP

- User onboarding with phone and email verification
- Government ID verification flow (Aadhaar-based workflow with consent)
- Gender-based matching preferences
- Room listing and roommate profile cards
- In-app chat with report/block options

## Safety Features

- Mandatory consent before document upload
- Data encryption at rest and in transit
- Panic report and abuse report workflow
- Profile trust score (verification + behavior signals)

## Compliance Notes

- Follow India data privacy and identity verification laws
- Avoid storing sensitive data unless required
- Use tokenized verification via approved providers where possible

## Revenue Model

- Premium visibility for profiles
- Listing promotions for owners
- Subscription for advanced filters

## Milestones

1. Validation interviews and policy review
2. MVP app with verification + matching
3. Safety moderation tools
4. Pilot launch in one city
