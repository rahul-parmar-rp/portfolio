# Instagram Mass Management Extension

## Description

This extension allows users to export a list of names and people they followed on Instagram. It enables mass restrictions and un-following, similar to a bookmarking feature for easy management of followed accounts.

## Plan

- Analyze Instagram's API for extracting follower data.
- Create a user interface for exporting and managing followers.
- Implement features for mass restriction and un-following.
- Conduct user testing to ensure functionality and ease of use.
- Prepare documentation for users and submit the extension to the Chrome Web Store.
