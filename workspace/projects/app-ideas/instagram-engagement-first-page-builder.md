# Draft Plan: Instagram Engagement-First Page Builder

## Idea Summary

Design an Instagram Business page setup and posting framework focused on engagement first, not vanity metrics.

## Problem

Most new pages optimize for follower count too early and miss repeat interactions that build long-term community.

## Core MVP

- Niche definition worksheet (one page)
- Content pillar planner (3 to 5 pillars)
- Caption templates for comments and saves
- Weekly posting plan:
  - 3 reels
  - 2 carousels
  - daily stories
- CTA library for comments and DMs

## Engagement-First Rules

- Every post asks one clear conversation question
- Reply to early comments within first 30 minutes
- Story polls used daily for feedback loops
- Reuse top-performing hooks in next week plan

## MVP Tooling Options

- Option A: Notion/Sheet + manual posting
- Option B: Node scheduler + Meta Graph API for approved posts
- Option C: Semi-auto with caption suggestions and manual publish

## What To Track Weekly

- comments per post
- saves per post
- shares per post
- profile visits from content
- repeat commenters

## Milestones

1. Create niche, pillars, and page positioning kit
2. Build 30-day post bank
3. Add engagement analytics board
4. Automate approved content scheduling

## MVP Success Metrics

- 20% growth in comments week over week
- Repeat commenters in top 20 audience segment
- Higher saves-to-impressions ratio after 30 days
