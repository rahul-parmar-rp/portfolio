# Draft Plan: Google Photos Wrapper

## Idea Summary

Create a wrapper app around Google Photos workflows for account management, organization, and bulk operations support.

## Problem

Power users with many albums and accounts need better organization and operational visibility.

## Core MVP

- Multi-account profile registry
- Album-level overview and quick links
- Task queue for cleanup actions
- Activity notes and reminders

## Compliance Notes

- Use official OAuth and APIs only
- Respect Google API quotas and permissions
- Keep user data secure and consent-driven

## Milestones

1. Auth and account switching architecture
2. Album and media summary dashboard
3. Batch task planner
4. Export and backup tracking
